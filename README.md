## Description

Ad Campaigns

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Local Services Required

```
Mysql > (greater than) 5.5.5 - For storing the campaigns in DB

Redis - For storing fetched campaigns in cache

PM2 - For managing the app process
```
